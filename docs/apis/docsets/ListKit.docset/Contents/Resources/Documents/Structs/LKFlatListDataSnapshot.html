LKFlatListDataSnapshot

@available(iOS 13.0, *)
public struct LKFlatListDataSnapshot<ItemIdentifierType>: LKListDataSnapshot
where
    ItemIdentifierType: Hashable, ItemIdentifierType: Sendable

A snapshot of the data for a flat list. This struct conforms to LKListDataSnapshot and is available on iOS 13.0 or later.
