LKTabView

@available(iOS 13.0, *)
@MainActor
public class LKTabView<ItemIdentifier>: UIScrollView, UIScrollViewDelegate
where
    ItemIdentifier: Hashable, ItemIdentifier: Sendable

A custom scroll view that represents a tab view in the ListKit framework.

LKTabView is a subclass of UIScrollView and conforms to the UIScrollViewDelegate protocol. It is designed to handle tab items identified by ItemIdentifier.

Parameters

ItemIdentifier

The type used to uniquely identify each tab item.
