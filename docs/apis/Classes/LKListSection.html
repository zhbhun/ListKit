LKListSection

public class LKListSection<SectionIdentifier, ItemIdentifier>
where
    SectionIdentifier: Hashable, SectionIdentifier: Sendable,
    ItemIdentifier: Hashable, ItemIdentifier: Sendable

A generic class representing a section in a list.

LKListSection is used to manage a section in a list, where each section has a unique identifier and contains multiple items, each with its own identifier.

Parameters

SectionIdentifier

The type of the unique identifier for the section.
ItemIdentifier

The type of the unique identifier for the items within the section.
