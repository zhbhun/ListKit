LKListCompositionalSectionSupplementary

@available(iOS 13.0, *)
public class LKListCompositionalSectionSupplementary<SectionIdentifier>: LKListSupplementary
where
    SectionIdentifier: Hashable, SectionIdentifier: Sendable

A class representing a supplementary view for a compositional section in a list.

LKListCompositionalSectionSupplementary is a generic class that conforms to LKListSupplementary. It is used to define supplementary views for sections in a compositional layout.
