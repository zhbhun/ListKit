LKListCompositionalGroup

@available(iOS 13.0, *)
public class LKListCompositionalGroup<ItemIdentifier>: LKListCompositionalItem<ItemIdentifier>
where
    ItemIdentifier: Hashable, ItemIdentifier: Sendable

A compositional group that represents a collection of items in a list.

LKListCompositionalGroup is a subclass of LKListCompositionalItem that groups multiple items together. It is used to define the layout and organization of items within a list.

Parameters

ItemIdentifier

The type used to uniquely identify items in the group.
