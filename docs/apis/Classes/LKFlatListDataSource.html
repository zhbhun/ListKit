LKFlatListDataSource

@available(iOS 13.0, tvOS 13.0, *)
open class LKFlatListDataSource<ItemIdentifierType>: LKListDataSource
where
    ItemIdentifierType: Hashable, ItemIdentifierType: Sendable

A data source class for flat list that conforms to LKListDataSource. This class is available on iOS 13.0 and tvOS 13.0 or later.
